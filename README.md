# Wordle

A [wordle](https://www.nytimes.com/games/wordle/index.html) clone. Unlike the original 5-letter format, this rendition fetches words of varying lengths from an external API, ensuring a unique experience every game.

## Usage

Clone the repository:

```bash
git clone git@github.com:brysonbw/wordle.git
```

```bash
cd wordle
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
